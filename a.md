Есть двумерный массив размером N x 2.

Его можно записать в одномерный массив размером N * 2.
Обращаться к индексу в строке row и столбце col можно по формуле 2 * row + col.

```
1 2
3 4
5 6

   0           1           <- col
0  [0][0] = 1  [0][1] = 2
1  [1][0] = 3  [1][1] = 4
2  [2][0] = 5  [2][1] = 6
^
row

     row  col
[2 * 0  + 0  = 0] = 1
[2 * 0  + 1  = 1] = 2

[2 * 1  + 0  = 2] = 3
[2 * 1  + 1  = 3] = 4

[2 * 2  + 0  = 4] = 5
[2 * 2  + 1  = 5] = 6
```
